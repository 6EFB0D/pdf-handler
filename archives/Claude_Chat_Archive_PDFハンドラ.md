### Chat 1 ｜ PDFハンドラアプリの開発企画
**日付**: 2024年12月26日  
**URL**: https://claude.ai/chat/bebd5804-ef30-4b93-b970-29b5ab07bf08

#### 概要
ファイルサーバ上のPDFファイルを管理するデスクトップアプリ「PDFハンドラ」の開発企画からスタート。

#### 主な議論・決定事項
- **コア課題の特定**: プレビュー中のファイルロック問題（プレビューしながらリネームできない）
- **技術スタック決定**:
  - 言語: C# / .NET 6.0
  - UI: WPF (MVVM アーキテクチャ)
  - PDF表示: PDFium.NET
  - PDF操作: iText 7
- **ファイルロック解決策**: メモリストリームにPDFを読み込み、ファイルハンドルを即時解放
- **GUI設計**: 3ペインレイアウト（フォルダツリー / サムネイル一覧 / プレビュー）

#### 成果物
- `01_初期企画書.md` — プロジェクト全体の企画書
- `02_GUI詳細設計書.md` — 画面レイアウト・メニュー・ショートカット設計
- `README.md` — プロジェクト概要

### Chat 2 ｜ デスクトップアプリの実装開発
**日付**: 2024年12月27日  
**URL**: https://claude.ai/chat/e68223fb-3c69-4133-b4e8-90479c6b5e35

#### 概要
企画書をもとに WPF アプリケーションの実装を開始。

#### 主な議論・決定事項

  - App.xaml 競合によるUI起動エラーを修正
  - using文不足のビルドエラーを修正
- **成果物**: ビルドエラーゼロの動作可能アプリ

### Chat 3 ｜ PDFハンドラ v3.4.2
**日付**: 2024年12月27日  
**URL**: https://claude.ai/chat/2021d228-c1db-4ec3-8c36-aa549eec9519

#### 概要
v3.4.2 のリリース準備中に、iText 7 のライセンス問題が浮上。

#### 主な議論・決定事項
- **バグ修正**: お気に入り設定ダイアログのOK/Cancelボタンが切れる問題
  - `AddFavoriteDialog.xaml` ウィンドウ高さ 250px → 300px に変更
- **配布方法の検討**:
  - ZIPファイル配布（初心者向け・推奨）
  - インストーラー作成（上級者向け）
  - ビルドスクリプト・READMEを作成
- **⚠️ 重大発見 — ライセンス問題**:
  - iText 7 は AGPL v3.0 ライセンス
  - 商用利用には商用ライセンスが必要（約 **150,000円/年**）
  - 対策として PdfSharp（MITライセンス・完全無料）への移行を推奨
- **成果物**: リリースパッケージ（ライセンス文書・Aboutダイアログ・ビルドスクリプト）

### Chat 4 ｜ PDF handler 4.0 フリーライセンス化
**日付**: 2024年12月30日  
**URL**: https://claude.ai/chat/9fc51aec-5e2a-4d04-bec7-f9fbfe20f5ba

#### 概要
iText 7 → PdfSharp 6.1.1 への移行を実施。**v4.0.0** として大幅アップグレード。

#### 主な議論・決定事項
- **ライブラリ移行**: iText 7 (AGPL v3.0) → **PdfSharp 6.1.1 (MIT License)**
  - 商用利用コスト: 年間150,000円 → **¥0（完全無料）**
- **実装内容**:
  - PdfSharp API を使用した PDF 結合・分割サービスを再実装
  - バージョン情報をハードコードから **Assembly 属性による動的取得**に変更
  - 自動ビルド番号インクリメント実装
- **技術的問題解決**:
  - PdfSharp の `ImportMode` ドキュメント処理エラー
  - プロパティ名の大文字小文字ビルドエラー
  - XAML の重複コントロール名競合
- **GitHub リポジトリ整備**:
  - URL: https://github.com/kwgsmktgoplan/pdf-handler
  - プライバシーポリシー・利用規約・ライセンス情報を作成
- **セマンティックバージョニング**採用（MAJOR.MINOR.PATCH 形式）
- **成果物**: 17ファイルの完全実装パッケージ
