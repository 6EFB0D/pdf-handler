■ 1. はじめに

PDFハンドラへようこそ！

PDFハンドラは、ファイルサーバ上のPDFファイルを効率的に管理・編集する
ためのデスクトップアプリケーションです。

【主な機能】
• PDFファイルのプレビュー表示
• プレビュー中でもファイル名を変更可能
• 複数PDFファイルの結合
• PDFファイルの分割
• ヘッダ・フッターの挿入（文書タイトル・ページ番号）
• お気に入りフォルダの管理
• サムネイル/リスト表示の切り替え

【特徴】
• 直感的な3ペイン構成のインターフェース
• ファイルロック問題を解決（プレビュー中でも名前変更可能）
• 買い切りライセンス（Standard版）でご利用いただけます

■ 2. 動作環境

【必須環境】
• OS: Windows 10 / Windows 11（64bit）
• .NET Runtime: .NET 6.0 以上
• メモリ: 4GB以上推奨
• ディスク空き容量: 100MB以上

【推奨環境】
• メモリ: 8GB以上
• ディスプレイ: 1920x1080以上

■ 3. インストール方法

【インストーラ（setup.exe）— 通常はこちら】
1. 製品ページ（https://office-goplan.com）または GitHub Releases から最新版を開く
2. Assets の PdfHandler-<バージョン>-prod-setup.exe をダウンロード
   例: PdfHandler-1.1.3-prod-setup.exe
3. ダウンロードしたファイルを実行し、画面の指示に従ってインストール
4. スタートメニューまたはデスクトップのショートカットから起動

【ZIP 版（インストーラがブロックされる場合のみ）】
セットアップ EXE が SmartScreen や Defender で止まる場合:
1. 同じ Releases の PdfHandler-<バージョン>-win-x64.zip をダウンロード
   例: PdfHandler-1.1.3-win-x64.zip
2. .zip を右クリック →「すべて展開」
3. 展開フォルダ内の PdfHandler.UI.exe をダブルクリックして起動

※ 通常の利用は setup.exe をご利用ください。

【チェックサム（任意）】
• 同梱の *-checksum.txt に SHA-256 が記載されています
• PowerShell 例: Get-FileHash -Path .\ファイル名 -Algorithm SHA256

【初回起動時】
• Windows Defender の警告が表示される場合があります
• 「詳細情報」→「実行」をクリックしてください

■ 4. 基本的な使い方

│  画面構成                               │
├──────────┬──────────┬─────────────────┤
│ ツリー   │ ファイル │ プレビュー      │

【4-1. フォルダを開く】

方法1: ツールバーから
1. ツールバー左端の「📁 フォルダを開く」ボタンをクリック
2. フォルダ選択ダイアログが表示されます
3. PDFファイルがあるフォルダを選択
4. 「フォルダーの選択」をクリック

方法2: メニューから
1. メニューバーの「ファイル」→「フォルダを開く」
2. フォルダを選択

【4-2. PDFをプレビュー表示】

1. 左側のツリーグリッドでフォルダを選択
2. 中央のファイルリストにPDFファイルが表示されます
3. ファイルをクリックすると、右側にプレビューが表示されます

【プレビュー操作】
• ←/→ キー: ページ移動
• Ctrl + マウスホイール: ズーム
• マウスドラッグ: スクロール

【4-3. ファイル名を変更】

方法1: F2キー
1. ファイルを選択
2. F2キーを押す
3. ファイル名を編集
4. Enterキーで確定

方法2: 右クリックメニュー
1. ファイルを右クリック
2. 「名前変更」を選択
3. ファイル名を編集
4. Enterキーで確定

【重要】
• プレビュー表示中でもファイル名を変更できます
• ファイルロック問題を解決しています

【4-4. 表示モードの切り替え】

サムネイルビュー:
• ツールバーの「🖼️」ボタンをクリック
• PDFの1ページ目をサムネイル表示

リストビュー:
• ツールバーの「📋」ボタンをクリック
• ファイル名、サイズ、更新日時を一覧表示

【4-5. プレビューのON/OFF】

• ツールバーの「👁️」ボタンをクリック
• F12キーでも切り替え可能
• プレビューOFF時はファイルリストが拡大表示されます

■ 5. PDF結合機能

複数のPDFファイルを1つのPDFファイルに結合します。

【手順】

1. 結合したいPDFファイルを選択
   • Ctrlキーを押しながらクリック（複数選択）
   • Shiftキーで範囲選択も可能

2. メニューから「ツール」→「PDFを結合」を選択
   または、ツールバーの「🔗」ボタンをクリック

3. PDF結合ダイアログが表示されます

4. 結合順序を調整（必要に応じて）
   • ファイルを選択して「上へ」「下へ」ボタンで移動
   • またはドラッグ&ドロップで並び替え

5. ファイルの追加/削除（必要に応じて）
   • 「ファイル追加」ボタンでファイルを追加
   • 選択して「削除」ボタンで削除

6. 保存先を指定
   • 「参照」ボタンをクリック
   • 保存先フォルダを選択

7. ファイル名を入力
   例: 結合済み_報告書.pdf

8. 「結合実行」ボタンをクリック

9. 処理完了後、保存先に結合されたPDFが作成されます

【注意事項】
• 結合処理中は他の操作を行わないでください
• 大量のファイルを結合する場合は時間がかかります
• 元のファイルは変更されません（新しいファイルが作成されます）

■ 6. PDF分割機能

1つのPDFファイルを複数のPDFファイルに分割します。

【手順】

1. 分割したいPDFファイルを選択

2. メニューから「ツール」→「PDFを分割」を選択
   または、ツールバーの「✂️」ボタンをクリック

3. PDF分割ダイアログが表示されます

4. 分割方法を選択

   【方法A: ページ範囲で分割】
   • 範囲を指定して分割
   • 例: "1-3, 4-7, 8-10" と入力
   • 結果: 3つのPDFファイルが作成されます

   【方法B: 1ページずつ分割】
   • すべてのページを個別のPDFファイルに分割
   • 10ページのPDFなら10個のファイルが作成されます

   【方法C: 等分割】
   • 分割数を指定（例: 3）
   • ページ数を均等に分割します

5. 保存先を指定
   • 「参照」ボタンをクリック
   • 保存先フォルダを選択

6. ファイル名規則を入力
   例: 報告書_[番号].pdf
   → 報告書_001.pdf, 報告書_002.pdf, ...

7. 「分割実行」ボタンをクリック

8. 処理完了後、保存先に分割されたPDFが作成されます

【注意事項】
• 元のファイルは変更されません
• ファイル名の[番号]部分は自動的に001, 002, ...に置き換えられます
• 既存のファイルがある場合は上書き確認が表示されます

■ 7. ヘッダ・フッター機能

PDFファイルにヘッダ（文書タイトル）とフッター（ページ番号）を挿入します。

【手順】

1. 対象のPDFファイルを選択

2. メニューから「ツール」→「ヘッダ・フッター」を選択
   または、ツールバーの「📋 ヘッダ・フッター」ボタンをクリック

3. ヘッダ・フッター設定ダイアログが表示されます

4. ヘッダー設定
   • 「ヘッダーを表示」にチェック
   • 文書タイトルを入力（例: 報告書2025）
   • 配置: 左／中央／右 を選択

5. フッター設定
   • 「フッターを表示」にチェック
   • ページ番号形式を選択
     - 1       : ページ番号のみ（例: 1, 2, 3）
     - 1/10    : 総ページ数付き（例: 1/10, 2/10）
     - -1-     : ハイフン囲み（例: -1-, -2-）
   • 配置: 左／中央／右 を選択

6. フォント・マージン（必要に応じて）
   • ヘッダー用フォント: フォント名・サイズ（6〜24pt）を個別に指定
   • フッター用フォント: フォント名・サイズを個別に指定
   • 推奨フォント: MS Gothic、Meiryo、Yu Gothic など日本語対応フォント
   • ヘッダーマージン／フッターマージン: ページ端からの余白（pt）

7. 「ページ編集時に自動再適用」にチェック（推奨）
   • ページの削除・挿入後に、ヘッダ・フッターを自動で再適用します

8. 「適用」ボタンをクリック

9. 処理完了後、元のPDFファイルにヘッダ・フッターが追加されます

【注意事項】
• 元のPDFファイルを直接変更します（上書き保存）
• 設定は %AppData%\PdfHandler\headerfooter.json に保存され、
  次回起動時も同じ設定が使われます
• ページの削除や挿入を行うと、自動再適用が有効な場合のみ
  ページ番号が正しく再計算・再描画されます

■ 8. お気に入りフォルダ機能

よく使うフォルダをお気に入りに登録できます。

【お気に入りに追加】

1. ツリーグリッドでフォルダを右クリック
2. 「お気に入りに追加」を選択
3. 名前を入力（デフォルトはフォルダ名）

【お気に入りから開く】

1. メニューバーの「表示」→「お気に入り」
2. 登録したフォルダが表示されます
3. クリックすると該当フォルダが開きます

【お気に入りの管理】

削除:
1. お気に入りを右クリック
2. 「削除」を選択

名前変更:
1. お気に入りを右クリック
2. 「名前変更」を選択
3. 新しい名前を入力

■ 11. よくある質問（FAQ）

【Q1】PDFをプレビュー表示中にファイル名を変更できますか？

A: はい、できます！PDFハンドラは、PDFをメモリに読み込んで表示する
   ことで、元ファイルをロックしません。プレビュー表示中でも
   ファイル名の変更が可能です。


【Q2】大きなPDFファイル（100MB以上）も扱えますか？

A: 可能ですが、メモリ使用量が増加します。設定でメモリ制限を
   調整することをお勧めします。また、プレビュー品質を「低」に
   設定すると、動作が軽くなります。


【Q3】結合・分割処理を途中でキャンセルできますか？

A: 現在のバージョンでは、処理開始後のキャンセルはできません。
   処理が完了するまでお待ちください。


【Q4】ネットワークドライブ上のPDFも扱えますか？

A: はい、ネットワークドライブ上のPDFファイルも扱えます。
   ただし、ネットワーク速度により動作が遅くなる場合があります。


【Q5】複数のPDFハンドラを同時に起動できますか？

A: はい、複数のインスタンスを同時に起動できます。
   ただし、同じファイルを複数のインスタンスで編集しないように
   注意してください。


【Q6】アップデート通知が表示されません

A: 起動時のお知らせを「次回以降表示しない」にしている場合は出ません。
   再有効化は「ヘルプ」→「バージョン情報」内の「起動時のお知らせを再び表示」（お知らせを止めているときだけ表示）です。
   今すぐ確認する場合は、同じ画面の「更新を確認」をご利用ください。
   「ヘルプ」→「バージョン情報」内の「更新を確認」で
   手動確認できます。自動ダウンロード・インストールは行われません。


【Q7】PDFのページ数が多いと動作が遅くなります

A: 設定で以下を試してください：
   • プレビュー品質を「中」または「低」に設定
   • メモリ使用量制限を調整
   • サムネイルサイズを「小」に設定


【Q8】エクスプローラからファイルをドラッグ&ドロップできますか？

A: プレビュー領域へPDFをドロップすると、表示中PDFの現在ページの前に
   挿入できます。ページ編集機能は有償版の機能です。


【Q9】デバイス（PC）のライセンスを解除（デアクティベーション）したい

A: ヘルプメニュー →「ライセンス」を開き、有効なライセンスが登録されている場合に
   表示される「デバイス管理」から、登録デバイス一覧の「解除」ボタンで
   該当デバイスのライセンスを解除できます。
   （例: PCを譲渡・廃棄する前に、別のPCで使えるようにする）

■ 12. トラブルシューティング

【問題1】アプリケーションが起動しない

原因と対処法:
• .NET Runtimeがインストールされていない
  → .NET 6.0 以上をインストール
  → https://dotnet.microsoft.com/download

• Windows Defenderがブロックしている
  → 「詳細情報」→「実行」をクリック

• 破損したファイルがある
  → 再度ダウンロードして展開


【問題2】PDFが表示されない

原因と対処法:
• PDFファイルが破損している
  → 他のPDFビューアーで開けるか確認

• ファイルが大きすぎる
  → メモリ制限を確認・調整

• アクセス権限がない
  → ファイルの読み取り権限を確認


【問題3】結合・分割処理が失敗する

原因と対処法:
• 保存先フォルダへの書き込み権限がない
  → 権限を確認、または保存先を変更

• ディスク容量が不足している
  → 不要なファイルを削除

• PDFファイルにパスワードがかかっている
  → パスワード保護を解除してから実行


【問題4】設定が保存されない

原因と対処法:
• %APPDATA%への書き込み権限がない
  → 管理者権限で実行

• ディスク容量が不足している
  → 不要なファイルを削除


【問題5】プレビューが遅い

対処法:
• プレビュー品質を下げる（設定→プレビュータブ）
• サムネイルサイズを小さくする
• プレビューをOFFにして必要な時だけON
