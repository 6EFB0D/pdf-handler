# 有償ライセンス契約 追加条項（契約書への記載案）

## 概要

本ドキュメントは、利用規約（TERMS_OF_USE.txt）またはライセンス契約書に追加すべき条項の文案です。5ライセンスパックおよび「最初のアクティベート日から起算」の仕様を明文化するために使用してください。

---

## 追加条項の文案

### 第〇条（有償ライセンスの契約期間）

1. 単一ライセンス（1ライセンス）のサブスクリプション契約の場合、契約期間は購入完了日から起算し、1年間とします。更新日以降は、お客様が解約されない限り、1年ごとに自動更新されます。

2. **複数ライセンスパック**（2ライセンス以上のまとめ買い）のサブスクリプション契約の場合、契約期間は**お客様が当該パックに含まれるライセンスキーのいずれか1つを最初にアクティベートした日**から起算し、1年間とします。

3. 前項の複数ライセンスパックにおいて、各ライセンスキーのアクティベート日が異なる場合であっても、契約期間および更新日は、最初のアクティベート日を基準としてパック全体で統一されます。

4. 複数ライセンスパックの有効期間は、購入日ではなく、最初のアクティベート日から開始するため、お客様はご都合のよいタイミングでライセンスキーを配布・アクティベートしていただくことができます。

---

### 第〇条（役務の提供時期）

1. 単一ライセンスのサブスクリプションについては、購入完了後、直ちにライセンスキーをご利用いただけます。

2. 複数ライセンスパックのサブスクリプションについては、**最初のライセンスキーのアクティベート日**を以って、当該パックに含まれる全ライセンスの役務提供が開始されます。購入後、アクティベート前の期間は、役務提供期間には含まれません。

---

### 第〇条（解約および返金）

1. サブスクリプションは、いつでも解約することができます。解約された場合、当該契約期間の満了日まではご利用いただけますが、満了日以降は自動更新されません。

2. 複数ライセンスパックを購入したが、いずれのキーもアクティベートしていない場合、購入日から〇〇日以内であれば、返金のご相談を承ります。（※返金条件は事業者様のポリシーに合わせて調整してください）

3. 解約方法は、［お問い合わせ先／解約専用ページ］にて案内します。

## TERMS_OF_USE.txt への組み込み例

利用規約に「有償ライセンスに関する特則」として、新たな条（例：第12条）を追加する場合の例です。

```
第12条（有償ライセンスの契約期間および役務提供）

1. 本ソフトウェアの有償ライセンス（サブスクリプション）の契約期間は、別途定める購入ページの記載に従います。

2. 複数ライセンスパック（2ライセンス以上のまとめ買い）の場合、契約期間は、当該パックに含まれるライセンスキーのいずれか1つをお客様が最初にアクティベートした日から起算し、1年間とします。各ライセンスのアクティベート日が異なる場合でも、契約期間および更新日は、最初のアクティベート日を基準としてパック全体で統一されます。

3. 役務の提供は、単一ライセンスの場合は購入完了後、複数ライセンスパックの場合は最初のアクティベート日に開始されます。
```

## 購入画面（Stripe Checkout等）での表示例

最終確認画面や商品説明に、以下のような文言を表示することを推奨します。

```
【5ライセンスパック サブスクリプション】

・5台分のライセンスキーをお届けします。
・有効期間：最初の1つのキーをアクティベートした日から1年間
・キーのアクティベート時期が異なっても、契約期間・更新日は最初のアクティベート日を基準とします。
・更新料：〇〇円/年（自動更新、いつでも解約可能）
・解約方法：［URL］
```

## 適用上の注意

1. 条項番号（第〇条）は、既存の利用規約の構成に合わせて調整してください。
2. 返金条件、解約方法の記載は、事業者様のポリシーに応じて具体化してください。
3. 本文案をそのまま使用する前に、必要に応じて専門家（弁護士）の確認を受けることを推奨します。
