## 個人情報を公開しないための共通ポイント

- **連絡先**: 個人メールではなく `support@yourdomain.com` などの事業者用メールを使用
- **領収証・領収書**: Stripe の自動領収証を活用し、発行者は事業者名のみ
- **問い合わせ**: フォーム経由で受け、送信先を support@ に転送
- **メール送信**: Resend 等で `noreply@` や `support@` から送信
