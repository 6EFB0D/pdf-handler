# Release v1.1.1 — PDFハンドラ

## ダウンロード

インストーラ（Windows 64bit）を GitHub Release の Assets から取得してください。

- `PdfHandler-1.1.1-prod-setup.exe`

## チェックサム（SHA-256）

検証例（PowerShell）:

```powershell
Get-FileHash -Path .\PdfHandler-1.1.1-prod-setup.exe -Algorithm SHA256
```

**期待値:**

```
61DF9AB62DF4EE03ECB13C753A9A1E3C55FEE6BD9868B9DE5F6D7A39B7E240E5
```

再ビルドした場合は、同梱の `PdfHandler-1.1.1-prod-setup-checksum.txt` の値を優先してください。

## 主な変更

- 銀行振込（日本）による購入対応
- 購入フロー: お支払いリンクをメールで案内
- 請求書・銀行振込時の領収同等リンクを購入完了メールに記載
- per-user インストーラ（`%LocalAppData%\Office Go Plan\PDFハンドラ\`）
- ライセンス・デバイス管理の改善

詳細は [CHANGELOG.md](https://github.com/6EFB0D/pdf-handler/blob/main/CHANGELOG.md) を参照してください。
