## 🚀 ビルド手順

### 必要な準備

1. **.NET 8.0 SDK** - 必須
2. **Docnet.Core ネイティブライブラリ** - 自動ダウンロード

### ビルドコマンド

```bash
# NuGet復元（Docnet.Coreダウンロード）
dotnet restore

# ビルド
dotnet build

# 実行
cd src/PdfHandler.UI
dotnet run
```

### 初回ビルド時の注意

Docnet.Coreは初回ビルド時にネイティブライブラリ（pdfium.dll等）を自動的にダウンロードします。

**ダウンロードサイズ:** 約15-20MB  
**所要時間:** 環境により30秒〜2分

## 🔧 トラブルシューティング

### 問題: PDFが表示されない

**原因1: Docnet.Coreがダウンロードされていない**
```bash
# 解決: クリーンビルド
dotnet clean
dotnet restore
dotnet build
```

**原因2: pdfium.dllが見つからない**
```bash
# 確認
ls src/PdfHandler.UI/bin/Debug/net8.0-windows/runtimes/

# pdfium.dll が存在するか確認
```

**解決策:**
```bash
# NuGet キャッシュをクリア
dotnet nuget locals all --clear

# 再ビルド
dotnet restore
dotnet build
```

### 問題: レンダリングが遅い

**解決策:**
1. サムネイルサイズを小さくする（デフォルト150x200）
2. プレビューDPIを下げる（デフォルト96）
3. 非同期処理を確認

### 問題: ビルドエラー

**エラー:** `Docnet.Core が見つからない`

**解決策:**
```bash
# 手動でパッケージをインストール
cd src/PdfHandler.Infrastructure
dotnet add package Docnet.Core --version 2.6.0
```
