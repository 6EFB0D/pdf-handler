# クイックスタート: .NET 8.0版

## ⚠️ 重要なお知らせ

**このバージョンは.NET 8.0で動作します。**

.NET 6.0のサポートは2024年11月に終了しました。  
セキュリティとパフォーマンスのため、.NET 8.0版をご使用ください。

## 🚀 5分で始める

### ステップ1: .NET 8.0 SDKのインストール

#### Windowsの場合
1. https://dotnet.microsoft.com/download/dotnet/8.0 を開く
2. **SDK 8.0.x (推奨)** をダウンロード
3. インストーラーを実行
4. 完了！

#### 確認
```bash
dotnet --version
# 8.0.xxx と表示されればOK
```

### ステップ2: ビルド

```bash
# プロジェクトディレクトリで実行
cd pdf-handler

# ビルド
dotnet restore
dotnet build
```

### ステップ3: 実行

```bash
cd src/PdfHandler.UI
dotnet run
```

## ✅ 完了！

アプリケーションが起動します。

## 💡 もし.NET 6.0版が必要な場合

git履歴から v2.1.1 をチェックアウトしてください。

```bash
git checkout v2.1.1
```

**ただし、セキュリティリスクがあるため推奨しません。**
