## 🎉 ハイライト

### ライセンスコスト完全削減を実現

PDFハンドラ v4.0.0では、PDF操作ライブラリを**iText 7（AGPL v3.0）から PdfSharp 6.1.1（MIT License）に完全移行**し、商用利用における年間ライセンスコスト（推定15万円以上）を**完全に削減**しました。

**移行の成果**:
- 年間ライセンス費用: 約15万円 → **¥0**
- 商用利用の制約: AGPL準拠必須 → **制約なし**
- ソースコード公開義務: 必須 → **不要**

これにより、企業・個人を問わず、**完全無償での商用利用**が可能になりました。

## ✨ 新機能

### 1. お気に入りフォルダ管理

頻繁にアクセスするフォルダを登録・管理できる機能を追加しました。

**機能詳細**:
- フォルダツリー上部に「お気に入り」専用セクション表示
- ドラッグ&ドロップまたは右クリックメニューで登録
- お気に入りから削除も簡単操作
- JSON形式でローカル保存、次回起動時に自動復元

**ユースケース**:
```
例: プロジェクトフォルダ、契約書フォルダ、請求書フォルダ等を
    お気に入り登録することで、毎回階層を辿る手間を削減
```

### 2. F2キーによるインライン編集

Excelライクな操作感でファイル名をその場で変更できます。

**機能詳細**:
- ファイル選択後、F2キーで即座に名前変更モードに移行
- Enter: 変更確定
- Esc: 変更キャンセル
- サムネイルビュー・リストビュー両方で動作

**従来との比較**:
```
従来: 右クリック → 名前変更 → ダイアログ → 入力 → OK
v4.0: F2 → 入力 → Enter

操作時間: 約5秒 → 約2秒（60%削減）
```

### 3. 動的バージョン管理

ビルド時に自動でバージョン番号とビルド番号を更新します。

**表示例**:
```
ヘルプメニュー → バージョン情報
「PDFハンドラ v4.0.0 (Build 1234)」

※Build番号は自動インクリメント
```

**サポート向上**:
- ユーザーからの問い合わせ時、正確なバージョン特定が可能
- GitHub Issuesでのトラブルシューティング効率化

## 🐛 修正した問題

### 致命的な問題

1. **PDF結合時のImportモードエラー**
   - 症状: 特定のPDFで結合時にクラッシュ
   - 原因: PdfSharp ImportモードとModifyモードの不適切な混在
   - 修正: Importモードに統一、AddPage処理を最適化

2. **PDF結合後の戻り値がnull**
   - 症状: 結合処理は成功するが、結果パスが取得できない
   - 原因: MergePdfsメソッドの戻り値処理が不完全
   - 修正: 明示的な戻り値設定、エラー時のnull返却

### 一般的な問題

3. **お気に入りフォルダの重複登録**
   - 症状: 同じフォルダが複数回登録される
   - 修正: 登録前の重複チェック処理を追加

4. **ファイル名変更時の検証不足**
   - 症状: 不正な文字を含むファイル名が受け入れられる
   - 修正: Windows無効文字の検証処理を追加

5. **起動時のお気に入り復元の不安定性**
   - 症状: まれに起動時にお気に入りが消える
   - 修正: JSON読み込み時の例外処理を強化

### UI/UX改善

6. **F2編集モード中のファイル切り替え**
   - 症状: 編集中に別ファイルをクリックすると変更が失われる
   - 修正: 未保存の変更がある場合は確認ダイアログ表示

7. **大量ファイル表示時のスクロール性能**
   - 症状: 1,000ファイル以上でスクロールがカクつく
   - 修正: ListView仮想化の最適化

## 📝 既知の問題

### 軽微な問題

1. **大容量PDF（100MB以上）のプレビュー遅延**
   - 影響: プレビュー表示に5秒以上かかる場合がある
   - 回避策: ページ数の少ないPDFで確認
   - 修正予定: v4.1.0（ページ単位読み込み実装）

2. **特殊文字を含むファイル名の表示**
   - 影響: 一部の絵文字や記号が正しく表示されない
   - 回避策: 標準的な文字のみ使用
   - 修正予定: v4.2.0（フォント対応強化）

### 機能制限

3. **PDF編集機能の未実装**
   - 現状: 注釈、ページ回転、透かし等は未対応
   - 予定: v4.5.0で実装（基本機能完成版）

4. **AI機能の未実装**
   - 現状: 文書要約、セマンティック検索等は未対応
   - 予定: v5.0.0で実装（Premium版）
