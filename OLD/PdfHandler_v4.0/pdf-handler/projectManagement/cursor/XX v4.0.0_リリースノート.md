# PDFハンドラ v4.0.0 リリースノート

**リリース日**: 2025年1月1日  
**バージョン**: 4.0.0

---

## 🎉 重要な変更

### 完全無償化を実現！

PDFハンドラ v4.0.0では、**完全無償化**を実現しました。

- **iText 7（AGPL v3.0）→ PdfSharp 6.1.1（MIT License）に移行**
- **年間ライセンス費用: 約15万円 → 0円**
- **商用利用の制限: あり → なし**
- **ソースコード公開義務: あり → なし**

これにより、企業・個人問わず完全に無償で商用利用できるようになりました。

---

## ✨ 新機能

### お気に入りフォルダ機能
- 頻繁にアクセスするフォルダを登録可能
- ツリービューの上部に専用セクション表示
- JSON形式で永続化

### F2キーによるインライン編集
- ファイル選択後、F2キーで即座に名前変更モードに
- Enterで確定、Escでキャンセル
- サムネイルビュー・リストビュー両方で対応

## 🔄 変更点

### PDF操作ライブラリの移行
- **PdfSharp 6.1.1**に完全移行
- PDF結合・分割機能をPdfSharpで再実装

### ライセンス体系
- MITライセンスによる完全無償化
- 商用利用も完全無償
- ソースコード公開義務なし

- メモリ使用量の削減（約20MB改善）

- ファイル名変更のUX向上
- フォルダツリーの操作性改善

- エラーメッセージの詳細化

## 🐛 修正点

- PdfSharp Importモードでのページ追加エラーを修正
- PDF結合後の戻り値処理を改善
- お気に入りフォルダの重複登録を防止
- ファイル名変更時の検証処理を改善
- 起動時のお気に入りフォルダ復元処理を安定化

## 📦 技術的詳細

### 使用ライブラリ
- **PdfSharp 6.1.1** (MIT License) - PDF操作・結合・分割
- **Docnet.Core 2.6.0** (MIT License) - PDFレンダリング・サムネイル生成
- **CommunityToolkit.Mvvm 8.2.2** (MIT License) - MVVMパターン
- **System.Drawing.Common 8.0.0** (MIT License) - 画像処理

### システム要件
- Windows 10/11
- .NET 8.0 Runtime
- Visual Studio 2022以上（開発時）
