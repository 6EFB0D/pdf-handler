# projectManagement フォルダ

このフォルダには、PDFハンドラプロジェクトの進捗管理とタスク管理に関するファイルが含まれています。

## ファイル一覧

### 進捗管理.md
週次進捗記録、マイルストーン管理、リスク管理を記録します。

### タスク管理.md
詳細なタスク一覧と各タスクの進捗状況を管理します。

### リリース前チェックリスト.md
リリース前に確認すべき項目のチェックリストです。

### v4.0.0_リリースノート.md
v4.0.0のリリースノートです。

## 使用方法

### 週次進捗確認
毎週金曜日に`進捗管理.md`を更新し、完了タスクをチェックします。

### タスク管理
新しいタスクを追加する場合は、`タスク管理.md`に追加します。
タスクが完了したら、ステータスを更新します。

### リリース準備
リリース前に`リリース前チェックリスト.md`のすべての項目を確認します。
