## 🚀 ビルド方法

### 方法1: Visual Studio（推奨）

```
1. PdfHandler.sln を開く
2. Releaseモードに変更
3. ビルド → ソリューションのリビルド
4. src\PdfHandler.UI\bin\Release\net8.0-windows\ 内のファイルをコピー
5. README.txt と LICENSE.txt を追加
6. ZIP圧縮
```

### 方法2: 自動ビルドスクリプト

```
1. build_release.bat をダブルクリック
2. 自動で以下が作成される:
   - PdfHandler_v3.4.2\ フォルダ
   - PdfHandler_v3.4.2.zip ファイル
   - README.txt と LICENSE.txt が自動コピー済み
```
