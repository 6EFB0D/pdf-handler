=====================================
PDFハンドラ v4.0.0
=====================================

ファイルサーバ上のPDFファイルを効率的に管理・編集するための
デスクトップアプリケーション

【重要】v4.0.0の変更点
✅ 完全無償化を実現！
✅ 商用利用が自由に可能！
✅ 年間ライセンス費用 約15万円 → 0円

=====================================
主な機能
=====================================

1. PDFプレビュー＆ファイル管理
   - フォルダ階層のツリー表示
   - PDFファイルのサムネイル/リスト表示
   - PDFプレビュー表示（ページめくり、ズーム機能）
   - ファイル名変更（F2キー対応）
   - お気に入りフォルダ登録

2. PDF結合
   - 複数のPDFファイルを1つに結合
   - 結合順序の調整（ドラッグ&ドロップ、上へ/下へボタン）
   - 進捗表示

3. PDF分割
   - ページ範囲指定分割（例: 1-3, 4-7, 8-10）
   - 1ページずつ分割
   - 等分割（例: 3分割）
   - ファイル名規則の設定

=====================================
動作環境
=====================================

【必須】
- OS: Windows 10 / 11 (64bit)
- .NET: .NET 8.0 Runtime（インストーラーに同梱）
- メモリ: 4GB以上推奨
- ディスク: 100MB以上の空き容量

【推奨】
- メモリ: 8GB以上
- ディスク: 500MB以上の空き容量（大容量PDF処理用）

=====================================
インストール方法
=====================================

1. ZIPファイルを展開
   PdfHandler_v4.0.0.zip → 任意のフォルダに展開

2. 実行ファイルを起動
   PdfHandler.UI.exe をダブルクリック

※インストーラーは不要です
※レジストリへの書き込みはありません

=====================================
使い方
=====================================

【基本操作】

1. フォルダを開く
   - ツールバーの「📁」ボタンをクリック
   - または メニュー → ファイル → フォルダを開く

2. PDFをプレビュー
   - サムネイル/リストから PDFをクリック
   - 右側のプレビューエリアに表示されます

3. ファイル名を変更
   - ファイルを選択して F2キーを押す
   - または 右クリック → 名前変更

【PDF結合】

1. 結合したいPDFファイルを選択（Ctrl+クリックで複数選択）
2. メニュー → ツール → PDFを結合
3. 結合順序を調整（必要に応じて）
4. 保存先とファイル名を指定
5. 「結合実行」ボタンをクリック

【PDF分割】

1. 分割したいPDFファイルを選択
2. メニュー → ツール → PDFを分割
3. 分割方法を選択
   - ページ範囲で分割: 範囲を入力（例: 1-3, 4-7, 8-10）
   - 1ページずつ分割: 自動で全ページを個別ファイルに
   - 等分割: 分割数を指定（例: 3分割）
4. 保存先とファイル名規則を指定
5. 「分割実行」ボタンをクリック

【お気に入りフォルダ】

1. フォルダツリーでフォルダを右クリック
2. 「お気に入りに追加」を選択
3. お気に入りセクションに表示されます

=====================================
キーボードショートカット
=====================================

Ctrl+O          フォルダを開く
F2              名前変更
F5              更新
F12             プレビューON/OFF
Delete          削除
Ctrl+A          すべて選択
Ctrl+D          選択解除
Ctrl+M          PDF結合
Ctrl+Shift+S    PDF分割

=====================================
トラブルシューティング
=====================================

【Q1】アプリが起動しない
A1: .NET 8.0 Runtimeがインストールされているか確認してください
    https://dotnet.microsoft.com/download/dotnet/8.0

【Q2】PDFのプレビューが表示されない
A2: 
    - PDFファイルが破損していないか確認
    - パスワード保護されていないか確認
    - F5キーで更新してみてください

【Q3】ファイル名を変更できない
A3:
    - ファイルが他のアプリで開かれていないか確認
    - フォルダの書き込み権限があるか確認

【Q4】PDF結合・分割が失敗する
A4:
    - PDFファイルが破損していないか確認
    - 保存先フォルダに書き込み権限があるか確認
    - ディスク容量が十分にあるか確認

【Q5】動作が遅い
A5:
    - メモリ不足の可能性があります
    - 他のアプリを閉じてみてください
    - 大容量PDFの場合は時間がかかる場合があります

=====================================
既知の制限事項
=====================================

- パスワード保護されたPDFは処理できません
- 一部の特殊なPDFフォーマットに対応していない場合があります
- 非常に大容量のPDF（500MB以上）は処理に時間がかかる場合があります
