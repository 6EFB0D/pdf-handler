# 変更履歴 (CHANGELOG)

## [4.0.0] - 2025-01-01

### 🎉 重要な変更

**完全無償化を実現！**

- **iText 7（AGPL v3.0）→ PdfSharp 6.1.1（MIT License）に移行**
- **年間ライセンス費用: 約15万円 → 0円**
- **商用利用の制限: あり → なし**
- **ソースコード公開義務: あり → なし**

これにより、企業・個人問わず完全に無償で商用利用できるようになりました。

### Changed (変更)

- PDF操作ライブラリをiText 7からPdfSharp 6.1.1に完全移行
- PDF結合機能をPdfSharpで再実装
- PDF分割機能をPdfSharpで再実装
- ライセンス体系を完全無償化（MITライセンス）

### Improved (改善)

- メモリ使用量の削減（約20MB改善）
- 配布パッケージサイズの削減（約3MB削減）
- PDF処理の安定性向上
- エラーハンドリングの改善

### Removed (削除)

- iText7パッケージ（AGPL v3.0ライセンス）
- iText7.pdfhtmlパッケージ

### Fixed (修正)

- PdfSharp Importモードでのページ追加エラーを修正
- PDF結合後の戻り値処理を改善
- リソース解放処理を最適化

## [3.4.2] - 2024-12-27

### Added (追加)

- お気に入りフォルダ機能
  - 頻繁にアクセスするフォルダを登録可能
  - ツリービューの上部に専用セクション表示
  - JSON形式で永続化
- F2キーによるインライン編集機能
  - ファイル選択後、F2キーで即座に名前変更モードに
  - Enterで確定、Escでキャンセル

### Improved (改善)

- ファイル名変更のUX向上
- フォルダツリーの操作性改善
- UI全体の応答性向上

### Fixed (修正)

- お気に入りフォルダの重複登録を防止
- ファイル名変更時の検証処理を改善
- 起動時のお気に入りフォルダ復元処理を安定化

## [3.0.0] - 2024-12-26
