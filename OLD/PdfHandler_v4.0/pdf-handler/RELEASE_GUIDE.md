## 📦 配布方法1: シンプル配布（推奨）

### ステップ1: 公開用ビルド

#### Visual Studioでビルド

1. **Visual Studio**でプロジェクトを開く
2. 上部のメニューから **「ビルド」** → **「ソリューションのリビルド」**
3. 完了を待つ

#### コマンドラインでビルド（オプション）

プロジェクトフォルダで以下を実行：

```bash
dotnet publish src/PdfHandler.UI/PdfHandler.UI.csproj -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
```

**パラメータ説明:**
- `-c Release`: リリースモード
- `-r win-x64`: Windows 64bit用
- `--self-contained true`: .NETランタイムを含める
- `-p:PublishSingleFile=true`: 1つのEXEにまとめる

## 🔒 注意事項

### .NETランタイムについて

**self-containedビルドの場合（推奨）:**
- .NETランタイムが含まれる
- ユーザーは何もインストール不要
- ファイルサイズは大きい（~100MB）

**framework-dependentビルドの場合:**
- .NETランタイムが含まれない
- ユーザーは.NET 8.0をインストール必要
- ファイルサイズは小さい（~数MB）

### セキュリティ警告

初回起動時、Windowsが警告を表示する場合があります：

```
「WindowsによってPCが保護されました」
```

**対処法:**
1. 「詳細情報」をクリック
2. 「実行」ボタンが表示される
3. クリックして起動

**理由:**
- デジタル署名がないため
- コード署名証明書が必要（有料、年間数万円）
