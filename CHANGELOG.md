# CHANGELOG

## [1.1.3] - 2026-05-23

### 追加
- 起動時の更新通知（自動ダウンロード・インストールは行いません）
- ライセンス購入前の「サーバーへの接続確認」
- GitHub Releases にインストーラ（setup.exe）と ZIP 版の両方を掲載（各 SHA-256 チェックサム付き）

### 改善
- フォルダツリーのドラッグ＆ドロップ（ドロップ先の解決・ハイライト表示）
- 更新確認・起動時お知らせの再有効化を「ヘルプ → バージョン情報」に集約

### 修正
- フォルダツリーでサブフォルダへ移動したつもりが親フォルダに入ってしまうことがある不具合

### 開発者向け（本リポジトリのソース閲覧者向け）
- Supabase #45329 向けの準備（新規テーブル GRANT+RLS ひな型、Security Advisor 初回確認）
- 公開向けドキュメント・表記を v1.1.3 に整合

## [1.1.1] - 2026-05-06

### 追加
- 銀行振込（日本）による購入対応（Stripe JP Bank Transfer）
- 購入フロー: お支払いリンクをメールで案内する方式
- 請求書・領収書リンクをライセンス発行メールに自動添付
- インストーラ形式での配布開始（Inno Setup / per-user インストール）

### 改善
- ライセンスキーの正規化処理を強化（4 桁区切り入力に対応）
- オフライン検証（HMAC）とオンライン検証のハイブリッド方式
- デバイス管理画面でデバイス名の編集・削除が可能に

### 修正
- 複数デバイスでのアクティベーション上限チェックの精度向上
